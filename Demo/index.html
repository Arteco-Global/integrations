Arteco integration examples

This is the API integration document for the Arteco Video Analytics system. The REST API of Arteco is used by Arteco's web UI to configure the backend and to provide information to the Omnia Web frontend. The Arteco REST API allows the user to send events, get server information and change the Arteco Server configuration (channels, peripherals and devices).
